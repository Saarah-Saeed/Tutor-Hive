Tutor Hive

Tutor Hive is a responsive educational platform built using HTML, CSS, and JavaScript. It helps students find suitable tutors while enabling teachers to connect with the right students through an intuitive and user-friendly interface.

Technologies Used:

HTML5
CSS3
JavaScript
